# 仿写微信跳一跳[H5]

引擎：three.js

语言：javasrcipt

介绍：仿制微信跳一跳的网页版，实现了跳一跳的基本核心功能

展示：https://jsmask.github.io/jump-game/index.html

![Image Game](http://pic.yupoo.com/jsmask/30c23976/f719fa06.png)

```

```

